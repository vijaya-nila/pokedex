 Pokédex Web App

A simple and responsive Pokédex web application built using **HTML, CSS, and JavaScript**.  
This project fetches Pokémon data from the **PokeAPI** and displays them in a clean card layout with search functionality.


 🚀 Features

- 🔎 Search Pokémon by name
- 🔢 Search Pokémon by number (ID)
- 📱 Responsive design (mobile + desktop)
- 🎨 Clean Pokémon card UI
- ⚡ Fast API data fetching
- 🧠 Filter Pokémon dynamically


🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- PokeAPI (https://pokeapi.co/)
